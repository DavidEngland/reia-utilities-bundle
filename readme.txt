== Description ==

**REIA Utilities Bundle** is a comprehensive WordPress maintenance plugin that combines 8 essential utilities into one powerful interface. Originally developed for Real Estate Intelligence Agency (REIA), this plugin provides professional-grade WordPress optimization tools that rival premium alternatives.

= 🌟 Key Features =

**🗄️ Database Optimizer**
- Fix autoload performance issues (critical for sites with large autoloaded data)
- Clean post revisions, spam comments, and trash
- Remove orphaned metadata
- Optimize MySQL tables

**🧹 Cache Cleaner**
- Clear object cache, transients, and Elementor cache
- Reset OPCache and rewrite rules
- Regenerate thumbnails

**⚡ Autoload Fixer**
- Real-time autoload size monitoring
- Automatically detect and fix large autoloaded options
- Safe optimization with performance impact reporting

**🔌 Plugin Analyzer**
- Complete plugin inventory with size analysis
- Security vulnerability scanning
- Update monitoring and performance impact assessment

**📁 File Cleaner**
- Smart detection of unnecessary files (logs, temp, cache)
- Safe deletion with core file protection
- Custom cleanup patterns

**🔒 Security Scanner**
- WordPress core security checks
- File permission analysis
- User security audit with overall security scoring

**🚀 Performance Booster**
- Comprehensive site performance analysis
- Database and configuration optimization
- Actionable improvement recommendations

**💾 Backup Manager**
- Multiple backup types (full, files, database, uploads)
- Automated ZIP compression
- Secure backup storage and management

== Installation ==

= Automatic Installation =

1. Login to your WordPress admin panel
2. Navigate to Plugins > Add New
3. Search for "REIA Utilities Bundle"
4. Click "Install Now" and then "Activate"
5. Navigate to Tools > REIA Utilities to start using

= Manual Installation =

1. Download the plugin ZIP file
2. Login to your WordPress admin panel
3. Navigate to Plugins > Add New > Upload Plugin
4. Choose the ZIP file and click "Install Now"
5. Activate the plugin
6. Navigate to Tools > REIA Utilities

= Requirements =

- WordPress 5.0 or higher
- PHP 7.4 or higher
- MySQL 5.6 or higher
- ZipArchive PHP extension (for backup functionality)

== Frequently Asked Questions ==

= Is it safe to use on production sites? =

Yes! The plugin includes comprehensive safety measures including permission checks, core file protection, and selective operations. However, we always recommend creating a backup before running major optimizations.

= Will this plugin conflict with caching plugins? =

No, REIA Utilities Bundle is designed to work alongside popular caching plugins. The Cache Cleaner utility can actually help clear various cache types to ensure your caching plugins work optimally.

= How often should I run these utilities? =

- **Database Optimizer**: Monthly or after major content changes
- **Cache Cleaner**: Weekly or after plugin/theme updates
- **Security Scanner**: Weekly or after any site changes
- **File Cleaner**: Monthly
- **Performance Booster**: After optimization changes
- **Autoload Fixer**: As needed based on performance monitoring

= Can I undo changes made by the plugin? =

Some operations like database cleanup are permanent, which is why the plugin shows detailed previews and recommendations before execution. The Backup Manager utility helps create restore points before major changes.

= Does this work with multisite installations? =

The plugin is primarily designed for single-site installations. Multisite compatibility may be added in future versions.

= What happens if a utility operation fails? =

The plugin includes comprehensive error handling and will provide detailed error messages. Operations are designed to fail safely without breaking your site.

= Is technical support available? =

Yes! Support is available through the WordPress.org plugin support forum. Premium support may be available directly from Real Estate Intelligence Agency.
